PromptLogic User Guide

Welcome to the official user guide for PromptLogic, a Windows‑native teleprompter engine designed to work directly with Microsoft PowerPoint. PromptLogic reads your slide notes, scrolls them smoothly, and executes embedded dot‑commands that automate your presentation.

1. Introduction

PromptLogic is a lightweight teleprompter designed for presenters who build their scripts inside PowerPoint. Instead of juggling separate teleprompter apps, PromptLogic uses the notes you already write and adds automation through simple commands.

Key features:

  • Reads script directly from PowerPoint notes
  • Smooth, adjustable scrolling
  • Highlight band for easy reading
  • Automatic slide advancement using dot‑commands
  • Pause and stop commands embedded in your script

PromptLogic keeps your workflow simple: write your script in PowerPoint, add commands where needed, and present with confidence.

2. System Requirements

  • Windows 10 or Windows 11
  • Microsoft PowerPoint installed
  • Permission for PromptLogic to automate PowerPoint

PromptLogic communicates with PowerPoint using standard Windows automation. If PowerPoint asks for permission, choose Allow.

3. Getting Started

Launching PromptLogic

Start PromptLogic like any other Windows application. PowerPoint should be open with your presentation loaded.

Loading Your Script

PromptLogic automatically reads the notes for the current slide in PowerPoint. As you advance slides, PromptLogic loads the next set of notes.

Starting the Teleprompter

  1. Open your PowerPoint presentation
  2. Open PromptLogic
  3. Choose connect in the PromptLogic control panel
  4. Press Start to begin scrolling

The teleprompter will scroll the notes smoothly. Any dot‑commands in the script will be executed automatically.

4. Writing Scripts in PowerPoint Notes

Your script lives entirely inside the PowerPoint Notes pane. PromptLogic reads the text exactly as written and executes dot‑commands when it encounters them.

Guidelines:

  • Write your script normally
  • Place dot‑commands on their own line
  • Keep paragraphs short for easier reading
  • Test your script before presenting

Example Notes Section

Welcome to the presentation. Today we’ll cover three main topics.

.nextslide

This slide introduces the first topic. Speak slowly and clearly.

.pause

When ready, continue to the next section.

.stop

5. Dot Commands (v1)

Dot‑commands are simple instructions embedded in your script. They must appear on their own line and begin with a period.

.nextslide

Advances PowerPoint to the next slide automatically.

.pause

Pauses scrolling until you manually resume.

.pause(n)

Pauses scrolling the the number of milliseconds provided in (n)
ex. n = 1000 would pause for 1 second.

.stop

Stops scrolling entirely. Useful at the end of a section or presentation.

Example Usage

This is the end of the introduction.

.nextslide

Here begins the next slide's content.

6. Teleprompter Controls

  • Start – Begin scrolling
  • Pause – Temporarily stop scrolling
  • Resume – Continue scrolling after a pause
  • Speed – Adjust scroll speed
  • Highlight Band – Keeps your eyes centered on the active line

7. Tips for Smooth Presentations

  • Place dot‑commands on their own line
  • Use short paragraphs for readability
  • Test your script before presenting
  • Keep your speaking pace in mind when setting scroll speed
  • Use .pause before complex explanations

8. Known Limitations (v1)

  • Only works with PowerPoint notes
  • No file‑loading for standalone text scripts
  • No OBS integration
  • Only three dot‑commands supported

These limitations will be addressed in future versions.
