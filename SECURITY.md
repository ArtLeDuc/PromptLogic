# Security Policy

Thank you for taking the time to help improve the security of PromptLogic.

If you discover a security vulnerability, please **do not open a public GitHub issue**.  
Security issues should be reported privately so they can be investigated and resolved safely.

## Reporting a Vulnerability

Please email your report to:

**[ajleduc@gmail.com]**

Include as much detail as you can:
- A clear description of the issue
- Steps to reproduce it (if possible)
- Any relevant logs, screenshots, or proof‑of‑concepts

I will review your report as quickly as possible and follow up with next steps.

## Scope

This policy applies to:
- The PromptLogic application
- The PromptLogic scripting engine
- The WebView2 communication layer
- Any controller integrations (e.g., OBS)

## Responsible Disclosure

Please avoid:
- Posting vulnerabilities publicly
- Sharing exploit details before a fix is available
- Testing in ways that could disrupt other users

Thank you for helping keep PromptLogic safe for everyone.
